1. Общие положения

Настоящее пользовательское соглашение определяет порядок использования программного обеспечения SecOps Buddy и условия обработки данных, получаемых в процессе его работы.

Использование программного обеспечения означает согласие пользователя с условиями настоящего соглашения.

2. Назначение системы

SecOps Buddy предназначен для мониторинга изменений состояния безопасности вычислительной системы, включая контроль параметров, указанных в функционале системы.

Программное обеспечение используется исключительно в целях повышения видимости и контроля изменений конфигурации безопасности.

3. Обрабатываемые данные

В процессе работы система может обрабатывать следующие категории данных:

- системные пользователи и их права;
- параметры конфигурации безопасности операционной системы;
- сведения об открытых сетевых портах и запущенных сервисах;
- параметры конфигурации SSH;
- технические идентификаторы, необходимые для функционирования системы уведомлений (включая идентификатор администратора в Telegram).

⚠️ Система не обрабатывает и не собирает данные, не относящиеся к её функциональному назначению.

4. Персональные данные

Обработка данных осуществляется в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных».

Система:

- не собирает избыточные персональные данные;
- не обрабатывает специальные категории персональных данных;
- не осуществляет передачу данных третьим лицам.

5. Цели обработки данных

Обработка данных осуществляется исключительно в следующих целях:

- выявление изменений состояния безопасности системы;
- уведомление администратора о выявленных изменениях;
- обеспечение корректной работы программного обеспечения.

6. Хранение данных

Все данные:

- обрабатываются и хранятся локально на системе пользователя;
- не передаются на внешние серверы разработчика;
- не используются в иных целях, не предусмотренных функционалом системы.

7. Согласие пользователя

Пользователь подтверждает, что:

- использует программное обеспечение добровольно;
- ознакомлен с перечнем обрабатываемых данных;
- даёт согласие на их обработку в рамках настоящего соглашения.

Согласие предоставляется при первом запуске системы и может быть отозвано путём прекращения её использования.

8. Заключительные положения

Настоящее соглашение применяется исключительно в рамках использования программного обеспечения SecOps Buddy и не регулирует иные отношения.
