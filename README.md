# SecOps Buddy

**Linux Security Drift Monitor с Telegram-оповещениями**

---

## О проекте

SecOps Buddy — это агент для Linux-сервера и Telegram-бот, которые помогают отслеживать изменения важных параметров безопасности. Программа регулярно делает "снимки" состояния системы, позволяет видеть разницу между ними и сразу информирует в случае подозрительных изменений через Telegram.

---

## Ключевые возможности (MVP v1)

- **Контроль SSH:** отслеживание настроек PermitRootLogin, PasswordAuthentication, порта SSH и других критичных параметров  
- **Открытые порты и сервисы:** анализирует вывод `ss`  
- **Состояние и основные правила фаервола:** поддержка UFW и firewalld  
- **Пользователи с sudo:** сбор актуального списка  
- **Failed logins:** анализ auth.log или journalctl  
- **Доступные обновления безопасности:** базовая проверка

---

## Telegram-команды

- `/status` — показать краткий статус системы
- `/diff` — выводит изменения с прошлого сканирования
- `/report` — подробный отчёт

---

## Безопасность

- Поддержка allowlist по Telegram user_id
- Логирование всех действий
- Отчётный режим: нет удалённого шелла, только просмотр изменений и ограниченные автоматические сценарии
